# RESERVATION SERVICE
This is an API service to manage customer information, vehicle information, and reservation information.

## Ruby Version
2.6.3

## System Dependencies
 * rails
 * sqlite3
 * puma
 * bootsnap
 * rspec-rails
 * factory_bot_rails
 * faker
 * database_cleaner
 * capybara

## Configuration
Run `bundle install`

## Database Setup
Run `rails db:create:all`\
Run `rails db:migrate`\
Run `rails db:seed`

## Test Suite
Run `rspec spec/`

## Deployment Instructions
Run `rails s`
